# Jobr

You will need node.js installed and two terminals to run this application.

### Prerequisites

What things you need to install the software and how to install them

On one terminal

```
cd frontend
npm install
```

On the other terminal
```
cd server
npm install
```

### Running the project

One one terminal

```
cd frontend
npm start
```
On the other terminal
```
cd server
node server
```

### Running ESLint
```
cd frontend
npm run lint
```
